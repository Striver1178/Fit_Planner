# AI Fitness Planner Frontend UI

A modern AI-based Fitness Planner frontend UI built using only:

- HTML5
- CSS3
- Vanilla JavaScript

This project focuses on a clean, responsive, and professional dashboard-style fitness application interface with interactive frontend functionality and smooth user experience.

---

# Features

## Modern Dashboard UI
- Minimal dark theme
- Elegant card layouts
- Clean typography
- Professional spacing
- Responsive mobile-first design
